## 题目
给定一组不含重复元素的整数数组 nums，返回该数组所有可能的子集（幂集）。

说明：解集不能包含重复的子集。

**示例 1**
```
输入：nums = [1,2,3]
输出：
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```

## 代码（经典密码学）
```Java
class Solution {
    public List<List<Integer>> subsets(int[] nums) {
        int len = nums.length;
        int size  = 1 << len;
        List<List<Integer>> result = new ArrayList<>(size);
        for(int i = 0;i < size;i++){
            List<Integer> line = new ArrayList<>();
            for(int j = 0;j < len;j++){
                if((i & (1 << j)) > 0){
                    line.add(nums[j]);
                }
            }
            result.add(line);
        }
        return result;
    }
}
```
