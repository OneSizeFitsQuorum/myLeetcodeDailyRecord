## 题目
给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

**示例1**

![](static/59.jpeg)
```
输入：n = 3
输出：[[1,2,3],[8,9,4],[7,6,5]]
```

**示例2**
```
输入：n = 1
输出：[[1]]
```

**说明**
* 1 <= n <= 20


## 代码
```JAVA
class Solution {
    public int[][] generateMatrix(int n) {
        int[][] result = new int[n][n];
        int left = 0;
        int right = n - 1;
        int up = 0;
        int down = n - 1;
        int index = 0;
        while(true){
            for(int i = left;i <= right;i++){
                result[up][i] = ++index;
            }
            if(++up > down) break;
            for(int i = up;i <= down;i++){
                result[i][right] = ++index;
            }
            if(--right < left) break;
            for(int i = right;i >= left;i--){
                result[down][i] = ++index;
            }
            if(--down < up) break;
            for(int i = down;i >= up;i--){
                result[i][left] = ++index;
            }
            if(++left > right) break;
        }
        return result;
    }
}
```
